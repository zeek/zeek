============
Module Index
============

Zeek ships with the following script modules, i.e. collections of related scripts in
a common directory.

Modules in the ``base/`` directory are all loaded by default, while ones in
``policy/`` provide optional functionality and customization options that users
are free to add in accordance with their site's policy and preferences.
